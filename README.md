# Customer_Behavior_Analysis
data anlaysis showcasing customer behaviourusing Python Sql and PowerBi

Overview
This project demonstrates an end-to-end data analytics workflow using Python, SQL (MySQL), and Power BI. The goal is to analyze customer shopping behavior and generate actionable business insights through data cleaning, exploration, querying, and visualization.

Tools & Technologies

Python (Pandas, NumPy, Matplotlib/Seaborn)

MySQL (for data storage and SQL analysis)

Power BI (for interactive dashboards)

Project Workflow

1.Data Loading (Python)

Imported the dataset into Python using Pandas.

Verified structure, data types, and basic statistics.

2. Exploratory Data Analysis (EDA)

Analyzed distributions, trends, and patterns.

Identified missing values, duplicates, and outliers.

3. Data Cleaning

Handled missing values and incorrect data types.

Removed duplicates and treated outliers.

Prepared a clean dataset for analysis.

4.SQL Analysis (MySQL)

Loaded the cleaned data into a MySQL database.

Wrote SQL queries to calculate KPIs such as:

Top-selling products

Average purchase value

Discount usage percentage

Customer segmentation (New, Returning, Loyal)

5. Power BI Dashboard

Built an interactive dashboard to visualize:

Sales trends

Product performance

Customer segments

Impact of discounts

Enabled data-driven insights for business decisions.

Key Outcomes

Identified high-performing products and customer trends.

Segmented customers for targeted strategies.

Created a professional dashboard for stakeholders.

Strong understanding of the complete data analytics lifecycle

Hands-on experience with Python, SQL, and Power BI
